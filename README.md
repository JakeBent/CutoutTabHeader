# CutoutTabHeader

## Installation
  `InboxViewcontroller` is a sample use with a UIScrollView. 
  
  Can be used with any subclass of a UIScrollView.
  
  To run the sample project:
    1. clone the repo
    2. open `CutoutTabHeader.xcodeproj`
    3. hit `Run`
